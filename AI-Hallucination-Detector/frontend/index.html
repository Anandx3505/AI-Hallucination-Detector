How GAT Detects Hallucinations

A 6-stage pipeline from raw text to explainable hallucination detection

1

📝 Input & Response Generation

A biomedical question is sent to LLaMA-2 (13B), which generates 5 answer candidates — 1 correct and 4 potentially hallucinated responses using controlled sampling.

2

🔢 Semantic Embedding

Each answer is converted to a 768-dimensional vector using BERT, capturing deep semantic meaning and contextual relationships.

3

🎯 Contrastive Learning (SimCLR)

Embeddings are refined via SimCLR — pulling same-class answers together and pushing different-class answers apart in a 128-dim space.

4

🕸️ Graph Construction (k-NN)

A semantic similarity graph is built — each answer is a node, and edges connect answers with cosine similarity > 0.85. Truthful answers cluster together.

5

⚡ Graph Attention Network (GAT)

The GAT learns attention weights per edge — some neighbors matter more than others. This is the key differentiator from GCN (which treats all edges equally).

6

✅ Classification & Explanation

Each answer is classified as Correct, Partially Correct, or Hallucinated. Attention heatmaps provide explainability.
